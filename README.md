# mgmt6219-capstone-analytics
Capstone analytics project : Rise 1 Million leadership analytics dashboard and report

About the Project

This repository presents my capstone analytics project completed as part of MGMT-6219. The project was conducted through a Riipen industry simulation in collaboration with Rise 1 Million, an organization focused on leadership development and workplace engagement.
The objective of the project was to analyze survey data to identify leadership challenges, engagement patterns, and opportunities for data-driven decision making. The final outcome included a professional report, interactive Power BI dashboard, and live presentation designed for both technical and non-technical stakeholders.
